General Overview

This is the first part of my thesis, in which I fine tuned a Text Classification model to analyze the daily sentiment of bitcoin across social media (Twitter and Reddit). The base model is Cryptobert by ElKulako in which I fine tuned using manually labelled dataset and enriched using Back-Translation with Google Translate. In conclusion, a model with accuracy of 68% (Increased by 33%) is created.

Comprehensive Breakdown

An empirical study was conducted using four text classification model and dataset with various enriching method and tool, which are Back-Translation using Top 10 & 20 language with the highest BLEU score on Google Translate, Back-Translation using HelsinkiNLP, and EDA_NLP by Jason Wei. It is found that using Back-Translation using Top 10 language with the highest BLEU score on Google Translate created the best model over four that’s experimented on, but it doesn't mean it’s always the best over every case.
